# Two Pointer Technique

This is a versatile technique used in problem solving where we have to efficiently traverse the given array / sequential data structures 

- Involves maintaining two pointers that traverse the data structure in a coordinated manner
- The pointers adjust based on specific criteria allowing for efficient exploration and optimal TC 

> When ever there is a requirement to find two data elements satisfying a specific condition two pointer technique is adopted 

Examples : 

Detecting a Palindrome 
    - Two Pointers from either end 

Reversing an array in place
    - Two pointer either end . Swap elements in 
    
Pair with a given sum in Sorted Array 
    - Two pointers on either ends. Move the pointers appropriately based on the current sum 

## Which Problems match this pattern ?

- Linear Data Structure 
    Examples: Array, LinkedList or String 
- Process pairs: Process data elements at two different positions simultaneously
- Dynamic pointer movememnt: Both pointers move independently of each other according to certain conditions or criteria. Also might move in same or different direction
